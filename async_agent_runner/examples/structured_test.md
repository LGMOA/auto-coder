# 第一个任务：创建 Web 应用基础结构

创建一个基本的 React 应用，包含以下组件：

- 主页面组件
- 导航栏组件
- 页脚组件

要求：
- 使用 TypeScript
- 应用 Tailwind CSS
- 包含基本的路由设置

# 第二个任务：实现用户认证功能

在现有应用基础上添加用户认证功能：

## 登录功能

实现用户登录表单，包含：
- 用户名/邮箱输入
- 密码输入
- 记住我选项
- 登录按钮

## 注册功能

实现用户注册表单，包含：
- 用户名输入
- 邮箱输入
- 密码输入
- 确认密码输入
- 注册按钮

## 密码重置

实现密码重置功能：
- 忘记密码链接
- 邮箱验证
- 新密码设置
