# 013-AutoCoder如何查看Token消耗

> auto-coder >= 0.1.21 可用

可以执行如下命令：

```shell
auto-coder store --source_dir YOUR_PROJECT
```

注意，需要确保你的项目里有 .auto-coder 目录。

输出结果：

```
+-----------------+----------------------+--------------------------+
| project         |   input_tokens_count |   generated_tokens_count |
+=================+======================+==========================+
| ByzerRawCopilot |                51391 |                     3321 |
+-----------------+----------------------+--------------------------+
```
